# Holdfast Press — GitHub brand assets

Generated from the "Broken Binding" mark (`public/favicon.svg`) — an incomplete off-white
ring (`#EDE6D6`) on ink (`#1A1D24`). Regenerate with `sharp` from the SVG if the mark changes.

| File | Size | Use |
|---|---|---|
| `holdfast-avatar-1024.png` | 1024×1024 | **Org / profile picture** (GitHub, social) |
| `holdfast-avatar-512.png` | 512×512 | Smaller square avatar fallback |
| `holdfast-social-1200x630.png` | 1200×630 | **Repo social preview** / OpenGraph card |

## Uploading (GitHub has no API for these — web UI only)

- **Org avatar:** github.com/organizations/holdfast-press/settings/profile → *Profile picture* → upload `holdfast-avatar-1024.png`
- **Repo social preview:** each repo → **Settings → Social preview → Edit → Upload an image** → `holdfast-social-1200x630.png`

Everything else (org description, repo descriptions, homepage URLs, topics) is set via the GitHub API.
